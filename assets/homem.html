deltaBillingFramework

The brand new billing framework made for just anyone! 

This helps? ;)

The deltaBillingFramework builds upon the stable and robust base that DSMSAPL left. The integration of SQLalchemy along with more modifications, towards the creation of a generalized framework system.

Features

A lil bit about the shebang!

Open Source

Based on Python and made available for free on GitHub, just anyone can modify/ use the framework UNDER LICENSE.

Customizable

Just any part of the system can be customized to a particular name, use or speed!

Responsive

Coded in a dynamic schema, I aim to make this program as responsive as possible!

Universal Applications and Compactibility

Backed by SQLalchemy, this program can be run literally anywhere, without any constraints. Be it the cloud, a Windows device, Linux, Android mobile devices or even Apple iOS/ MacOS!
